MTH2006 Topic 5
Nonparametric statistics

5.1 Introduction

The main aim of this topic is to introduce nonparametric statistics. This is a branch of statistics where we aim to understand a process or processes from data without models that rely on parameters. We’ll use various examples along the way with which you’ll be familiar, and that will help make clear the distinction between parametric and nonparametric statistics. In addition to nonparametric statistics, Topic 5 will also introduce you to new aspects of R programming. Programming is both useful in practice, e.g. when we want to analyse data in real life, but is also useful for helping affirm theory taught and for thinking about addressing statistical problems efficiently.

Before we start on Topic 5, we’ll mostly be using ggplot2 for graphics, so make sure that it’s loaded each time you start an R session. 

library(ggplot2)

It’s also worth remembering that R’s help files are usually excellent. So, if you’re ever unsure how to use a function, start by checking its help file. For example, we can get help on the function mean() using 

?mean
## starting httpd help server ... done

The key pieces of information in a help file are Usage, how to use the function, Arguments, the arguments to the function, and Value, the value(s) that the function returns. Almost all functions’ help files end with an Examples section. These typically give considered demonstrations of how to use functions and what they can do.

5.1.4 Resources

These notes and the exercises cover everything on nonparametric statistics that can turn up on your MTH2006 examination. They will introduce various concepts, often by outlining the theory behind them and then showing examples of their application and how they can be implemented in R. Nonetheless, you may wish to supplement your knowledge by considering alternative descriptions. The following resources may be useful; refer to References at the end of this Topic.

  • Davison (2003, Ch. 2) covers empirical distribution functions, order statistics and histograms.
  • Bowman and Azzalini (1997, Ch. 1 & 2), Silverman (1986, Ch. 3) or Davison (2003, sec. 7.1.2) give good accounts of kernel density estimation.
  • Good (2013) covers permutation tests, while Hart (1997, sec. 6.3) and Davison (2003, Example 7.35) are useful for snippets.
  • Govindarajulu (2007), Sprent (2012) and Sheskin (2000) give comprehensive treatments on nonparametric tests; Davison (2003, sec. 7.3) outlines most covered here.
  • One of the best resources for learning about R is its ‘An Introduction to R’ manual, which is available in pdf and html formats from https://cran.r-project.org/manuals.html, along with various other useful manuals.
