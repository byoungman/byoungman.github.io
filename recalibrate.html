    recalibrate

    statistical recalibration of spatial processes

    about

    This R package allows spatial processes represented by model simulation to be recalibrated by accounting for measurements on the processes. The aim is to infer the true - yet unknown - values of the spatial process.

    citation

    Youngman, B. D. and D. B. Stephenson (2019). Spatial inference for hazard event intensities using imperfect observation and simulation data. Preprint available from http://empslocal.ex.ac.uk/people/staff/by223/youngman-stephenson_recalibration.pdf.
